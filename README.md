# Crypto Wallet

[Live Website](https://crypto-wallet-livid.vercel.app/)

## Stack
- Next.js (Frontend + Backend)
- React Context API (State Management)
- shadcn/ui (UI Styling)
- MongoDB (Database)
- Vercel (Deployment)
